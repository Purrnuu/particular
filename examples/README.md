# Examples

## 1) Vanilla (standalone global bundle)

Uses `dist/particular.global.js` directly in a plain HTML page.

```bash
npm run example:vanilla
```

Open:

- `http://localhost:8080/examples/vanilla/index.html`

## 2) React app (library import)

This is a minimal Vite React app that imports this package via `file:../..`.

```bash
npm run example:react
```

Open the Vite URL shown in terminal (usually `http://localhost:5173`).
