# Examples

Both examples include four demos: Click Burst, Screensaver, Text to Particles, and Image to Particles. Use the nav bar to switch between them.

## 1) Vanilla (standalone global bundle)

Uses `dist/particular.global.js` directly in a plain HTML page — no build step.

```bash
npm run example:vanilla
```

Open: `http://localhost:8080/examples/vanilla/index.html`

## 2) React app (library import)

Minimal Vite + React app that imports `particular` via `file:../..`.

```bash
npm run example:react
```

Open the Vite URL shown in terminal (usually `http://localhost:5173`).
