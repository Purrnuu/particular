- [ ] Boids-based flocking behaviour — implement Craig Reynolds' Boids algorithm as a new force type. Three steering rules: separation (avoid crowding neighbors), alignment (steer toward average heading), cohesion (steer toward average position). Expose as `FlockingForce` implementing `ForceSource` with configurable rule weights, neighbor radius, max speed, and max steering force. Should compose with existing forces (attractors, mouse force).
- [ ] Advanced WebGL shader effects — extend the WebGL2 renderer with GPU-computed visual effects: per-particle distortion fields (heat haze, refraction), GPU-driven trails via transform feedback, screen-space post-processing (bloom, chromatic aberration, motion blur), SDF shapes for resolution-independent rendering, fluid/smoke simulation via ping-pong textures. Each effect opt-in to preserve baseline performance.
- [ ] Library modularisation (base + modules) — restructure for tree-shakeable imports to reduce bundle size. Split into lean core (engine loop, particle, emitter, vector) and optional modules (image particles, shatter, container glow, mouse trail, screensaver, canvas renderer, WebGL renderer). Preserve current single-import DX as default, with granular imports as opt-in for size-conscious apps.
